Signature Dynamics of Protein Families

SignDy calculates the signature dynamics of families of proteins that share similar folds, but not necessarily similar sequences. Signature dynamics includes shared mode profiles, shared covariance between residue fluctuations, and their variations across family members.

API Features

You can use SignDy to

  • create PDB ensembles of protein families from database searches
  • evaluate signature dynamics profiles and cross-correlations
  • calculate shared global mode profiles for assessing conserved motions among family members
  • calculate departures from shared mode profiles for assessing protein-specific features
  • examine the spectral overlaps
    • between family members
  • generate sequence-, structure- and dynamics-based trees for analyzing their evolution
