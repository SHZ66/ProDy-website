SignDy for Signature Dynamics Calculations

SignDy calculates signature dynamics from ensembles of protein structures using elastic network models.

API Features

  • Creation of PDB ensembles of related proteins from database searches or a user defined list
  • Calculation of mode ensembles for comparison of dynamics across family members
  • Calculation of spectral overlaps for analysis of conserved modes across protein families
  • Calculation and plotting of signature dynamics profiles and cross-correlations
  • Calculation of sequence-, structure- and dynamics-based trees for analysis of differences between family members
Reference
Eyal E, Bahar I Toward a Molecular Understanding of the Anisotropic Response of Proteins to External Forces: Insights from Elastic Network Models Biophys J 2008 94(9):3424-3435.
